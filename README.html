Névé

Éco-Aventures à portée de rail 🏔️🚆

Névé est une application mobile haute-fidélité développée avec Expo & React Native. Elle permet de découvrir des itinéraires de randonnée exceptionnels accessibles directement en train depuis l'Île-de-France.

✨ Fonctionnalités Clés

🚆 100% Rail & Zéro Carbone
Itinéraires calculés au départ des gares TER/Transilien pour une expérience écoresponsable.
🎨 Mode Sombre & Clair Dynamique
Changement en temps réel via un tiroir de navigation ergonomique et stable.
🔍 Filtres de Difficulté
Recherche intelligente et tri par niveaux de difficulté (Facile, Modéré, Difficile).
📱 Interfaces Premium
Alignements géométriques rigoureux et micro-interactions fluides.

⚙️ Guide d'Installation

Étape 1 : Cloner le projet
git clone https://github.com/NathanLaure/neve.git
Étape 2 : Installer les dépendances
bun install

(Ou utilisez npm install si vous préférez npm).

Étape 3 : Lancer le serveur de développement
bun run start

(Vous pouvez vider le cache avec npx expo start --clear en cas de besoin).

Optionnel : Linter & Formater
bun run lint
